## 🔧 技术细节

### GRU 模型架构
```python
GRUReachability(
  输入：(B, 10, 28)
  
  input_proj: Linear(28→256) → LayerNorm → ReLU
  gru: GRU(256, 2 层) → 取最后隐藏状态
  shared: Linear(256→128) → LayerNorm → ReLU
  reach_head: Linear(128→28)
  
  参数量：~1.2M
)
```

### 支撑值计算
```
margin = support_pred - (support_directions @ current_state)
alarm = any(margin < -threshold)
```

### 数据格式
```json
{
  "state": {
    "qpos": [14 维],
    "qvel": [14 维],
    "action": [14 维]
  },
  "ground_truth": {
    "actual_danger": bool
  }
}
```
