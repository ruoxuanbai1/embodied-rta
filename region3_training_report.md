## 🔧 模块 1: 决策因子贡献度

**原理**: 泰勒展开一阶近似
```
φ_ij = qpos_j × ∂a_i/∂qpos_j  (贡献度)
S_logic = Σ(|φ_ij| for j in F_legal) / Σ(|φ_ij| for all j)
```

**学习结果**:
- 8 个动作模态
- 每模态 14 个动作维度
- 每动作 3 个关键状态特征 (F_legal)

**输出**: `F_legal_profiles.json`

## 🔧 模块 2: 激活链路

**原理**: 汉明距离检测激活模式异常
```
binary_act = (activation > 0).astype(int)  # 二值化
M_ref = vote(binary_acts)  # 标准掩码 (>50% 激活)
D_ham = sum(curr_binary != M_ref) / 512  # 汉明距离
```

## 🔧 模块 3: OOD 检测

**原理**: 马氏距离检测分布外状态
```
D_ood = mahalanobis(state, μ, Σ_inv)
```

**学习结果**:

## 🎯 独立预警机制

**重要**: Region 3 的三个模块**各自独立触发预警**，不是融合成一个分数！

```
┌─────────────────────────────────────┐
│ 模块 1: 决策因子贡献度                │
│ S_logic < 0.4 → 逻辑异常预警          │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│ 模块 2: 激活链路                     │
│ D_ham > threshold → 激活异常预警      │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│ 模块 3: OOD 检测                     │
│ D_ood > 15.62 → OOD 预警             │
└─────────────────────────────────────┘

干预逻辑：OR (任一触发 → 动作打折扣)
```

**干预措施**:
| 触发模块 | 措施 |
|---------|------|
| 任意 1 个 | velocity×0.4, omega×0.4, torque×0.6 |
| 多个同时 | 同上 (不叠加) |

这与 Region 1、Region 2 架构一致：
- **Region 1**: 物理硬约束 → 紧急刹车
- **Region 2**: 可达集预测 → 减速×50%
- **Region 3**: 感知异常 → 动作×40%
