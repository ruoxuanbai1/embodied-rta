## 🏗️ 方案设计

### 1. 支撑向量设计 (28 个物理意义方向)

```
支撑方向构成:
├─ 关节限位方向 (28 个) — 每个关节的 qpos 最小值/最大值方向
├─ 速度 - 位置耦合方向 (28 个) — 捕捉"位置接近限位 + 速度推向限位"
└─ 随机方向 (32 个) — 覆盖未建模的不确定性

最终：SVD 压缩到 28 个方向 (受限于状态维度 28)
形状：(28, 28)
```

### 2. 可达集生成 (蒙特卡洛包络)

```python
对于每个训练样本:
  输入：当前状态 x_t, 历史 10 步
  
  蒙特卡洛扰动 (30 次):
    ├─ 初始状态扰动：x_0 = x_t + N(0, σ_state)
    ├─ 速度扰动：v = v_t + N(0, σ_vel)
    └─ 简单动力学外推 50 步 (1 秒)
  
  支撑值计算:
    对于每个支撑方向 d_i (共 28 个):
      投影 = d_i · x (对所有轨迹、所有时间步)
      支撑值 h_i = max(投影)  ← 取最大值 (包络上界)
  
  输出：支撑值向量 (28,)
```

### 3. 模型架构 (中等规模)

```python
GRUReachability(
  输入：(B, 10, 28)
  
  编码器:
    Linear(28→256) → LayerNorm → ReLU → Dropout(0.3)
    GRU(256, 2 层) → 取最后隐藏状态
    Linear(256→128) → LayerNorm → ReLU
  
  头 1 - 可达集预测:
    Linear(128→28) → 28 个支撑值
  
  头 2 - 危险分类:
    Linear(128→64) → ReLU → Linear(64→1) → Sigmoid
  
  输出:
    support_pred: (B, 28)
    danger_logit: (B, 1)
)

参数量：~310K
样本/参数比：79:1 (24,500 样本 / 310K 参数)
```

### 4. 训练配置

```python
# 损失函数
loss = Huber(支撑值) + 5.0 × WeightedBCE(危险概率)
# 危险样本权重×5, 分类正样本权重×3

# 数据增强
训练时 50% 概率加 2% 高斯噪声

# 优化器
AdamW(lr=1e-4, weight_decay=1e-4)
Batch=128, Epochs=100
Scheduler: ReduceLROnPlateau (基于 Youden J)
```

- **架构**: GRU(256, 2 层) + 双头输出

- **输出**: 支撑值 (28 维) + 危险概率 (1 维)

### OOD 统计量 (优化后)
- **传统方法**: `mu_safe`, `sigma_inv_safe` (只用安全数据)
- **混合方法**: `mu_all`, `sigma_inv_all` (安全 + 危险)
- **双分布方法** (推荐): `mu_safe`, `sigma_inv_safe`, `mu_danger`, `sigma_inv_danger`
- **位置**: `/root/act/outputs/region3_detectors/ood_stats_retrained.json`

### 使用方法
```python
# 双分布 OOD 分数
D_safe = mahalanobis(x, mu_safe, sigma_inv_safe)
D_danger = mahalanobis(x, mu_danger, sigma_inv_danger)
D_ood = D_safe - D_danger  # 越大越可能是危险
```
